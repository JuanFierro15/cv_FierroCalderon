03 · Proyecto

Trabajo destacado

2026 Proyecto Personal

Sistema de Gestión ICFES

Aplicación web MVC desarrollada en Java 21 con Spring Boot para la administración integral de instituciones preparatorias para el examen ICFES. Sistema completo con control de acceso por roles, gestión académica y funcionalidades avanzadas de automatización.

  • Control de acceso basado en roles (admin, docente, estudiante) implementado con Spring Security 6
  • Gestión de asistencias, generación de simulacros en PDF y materiales de estudio
  • Logs de auditoría y cierre automático de instituciones mediante tareas programadas
  • Internacionalización completa en 4 idiomas (español, inglés, francés, italiano) con Spring MessageSource
  • Base de datos PostgreSQL gestionada con ORM Hibernate vía Spring Data JPA
Java 21 Spring Boot Spring Security 6 Spring Data JPA Thymeleaf PostgreSQL Hibernate Maven i18n (4 idiomas)
